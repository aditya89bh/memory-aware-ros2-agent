# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful and harassment-free
experience for everyone.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experience levels.
- Give and receive constructive feedback.
- Focus discussion on what is best for the project and its users.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or imagery.
- Trolling, sustained disruption, or bad-faith participation.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and
pull requests that do not align with this code of conduct.
