## Positioning

`memory-aware-ros2-agent` is a Python ROS2 package foundation for recording
memory events, persisting task traces, and recalling relevant past workflow
context.

The repository should be positioned as infrastructure for robot workflow memory,
not as a planning framework, control stack, or general-purpose agent runtime.

## Audience

- Robotics engineers building ROS2 workflows
- Researchers exploring task memory and recall
- Developers building debuggable robot task execution systems
