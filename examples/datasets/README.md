# Example Datasets

This directory contains small, deterministic JSON datasets for demos, CLI smoke
tests, and documentation walkthroughs.

## Included Scenarios

- `pick-and-place`: part detection, placement, and successful completion.
- `cnc-tending`: CNC load, cycle monitoring, and unload workflow.
- `navigation`: route execution with an obstacle-driven reroute.
- `failure-recovery`: failed docking attempt followed by retry and recovery.

The samples are intentionally compact so they can be inspected by hand and used
with the `memory-inspect`, `memory-export`, and `memory-import` commands.
